# Issue 3: Formative Assessment Design

## Description
Develop a formative assessment (e.g., an exit ticket or a quick quiz) that checks for understanding of a key concept from the first lesson.

- A formative assessment document is created in `assessments/formative-assessment-1.md`.
- The assessment is clearly linked to Lesson 1.1 learning objectives.

- The assessment is designed to be quick and easy to administer.
